Helm

в нём не было сервисов:

pushgateway;
node-exporter;
alertnamager;
были только следующие поды:

prometheus-server;
prometheus-kube-state-metrics.
В качестве ответа прикрепите в форму ниже архив (zip, rar) со скриншотом списка подов и файлом values.yaml.
